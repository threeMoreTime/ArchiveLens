# ADR 0002 - 完整离线原生运行时

- 状态：**已采纳**
- 日期：2026-07-15
- 适用平台：Windows 10/11 x64

## 背景

依赖宿主安装 Tesseract、中文模型和 DjVuLibre 会使全新或断网电脑无法完整扫描。运行时下载也会增加首次使用失败、版本漂移与供应链风险。

## 决策

- Setup 与 Portable 都内置锁定版本的 Tesseract 5.5.0、四个 `tessdata_fast` 中文模型和 DjVuLibre 3.5.29。
- 构建期按 `scripts/native-dependencies.lock.json` 下载并校验 SHA-256；最终用户运行时不联网。
- 生产模式强制使用 `process.resourcesPath` 下的组件，不回退到系统 PATH；开发模式保留显式环境变量覆盖。
- DjVuLibre 继续作为独立命令行进程调用。分发包同时携带 GPL-2.0 文本与对应源码归档，发布前必须人工复核许可证义务。
- 发布清单记录版本、来源、许可证、原始制品哈希、运行时文件哈希与目录树哈希。

## 影响与回滚

安装包和安装后占用会增加，但全新断网环境可直接扫描全部支持格式。若需回滚，移除 `extraResources` 中的原生资源及生产路径注入即可恢复宿主依赖策略；不涉及数据库或用户数据迁移。
