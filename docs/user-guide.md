# 用户使用说明

## 创建检索任务

1. 打开“新建扫描”，选择整个本地文件夹、单个文件，或最多 200 个 PDF、DJVU、DJV、TIFF、JPEG、PNG 文件组成的清单。多个文件可来自不同目录并可混合格式；重复文件会自动去重。
2. 输入 1～32 个 Unicode code point 的单个文字或连续词语。
3. 如需处理古籍竖排，先在“设置 → 校对显示与扫描上下文”中选择阅读方向和关键词每侧 1～50 字的上下文长度，再点击“开始扫描”。出处页始终按源文件无损显示；上下文默认“从左到右、每侧 15 字”，任务页可使用原来源重新创建任务并继承该任务的单独设置。
4. 在任务页观察文件数、页进度、命中数和失败明细。总页数尚未统计完成时，界面显示不确定进度，不再显示误导性的 `N/0`。
5. 扫描完成后进入校对工作台，在完整页面缩略图和固定整页画布中查看词语高亮、上下文、系统判断和人工结论。每次切换命中时，中央出处页默认以 100% 居中显示；画布支持滚轮缩放、拖拽移动、手动适应窗口和重新居中，最大 400%。右上角“朝上、朝右、朝下、朝左”按钮可按源文件保存展示方向，同一源文件的所有命中共用且重开应用后仍会恢复；方向只影响中央出处页和高亮的显示，不旋转左侧结果缩略图，也不修改原文件、OCR 坐标或导出内容。PDF 会按显示物理像素从原文件重新渲染，位图和 DjVu 在超过 100% 时仅放大观察并显示提示。缩放不会推动工作台或产生外层纵向滚动条。右侧校对摘要可收起为进度窄栏，应用会记住本机上的展开状态。
6. 按“确认命中 / 需要复核 / 拒绝命中”完成判断，再使用“上一条 / 下一条待处理 / 下一条”继续浏览。备注停顿约 700ms 后自动保存；切换结果、分页或进入导出中心前也会提交当前草稿，`Ctrl+Enter` 可立即保存。数据库写入失败时，草稿会暂存于本机并阻止工作台内的切换操作，返回该结果后可继续重试。
7. 在导出中心选择 JSON 或离线 HTML，再点击一次主按钮导出。HTML 审阅报告会逐页验证并嵌入源文件派生的无损 PNG；PDF 至少使用 384 DPI。任一页面无法验证或生成时，本次导出会失败且不会覆盖已有报告；导出前需要确认无损大图片报告可能超过 300MB。扫描或校对未完成时，确认区还会同时说明阶段性报告的完整性风险。

HTML 审阅报告按命中页面分组，同一页图片只嵌入一次，并在淡红色背景上标出该页全部命中。报告支持文件和校对状态筛选、文件名/相对路径/命中词/OCR 上下文/备注搜索、结果排序、每页 10/20/50/100 张页面卡片、大图前后切换、返回顶部以及 A4 纵向打印。打印始终输出完整报告，不受当前筛选和分页影响。报告只显示文件名和任务内相对路径，不包含源文件绝对路径，也不依赖 ArchiveLens 正在运行。

检索词仅移除首尾 ASCII SPACE（U+0020），随后执行 NFC 规范化。匹配区分大小写，支持同一 OCR 行内的重叠精确字面匹配；不支持正则、通配符或跨 OCR 行匹配。任务创建后不能修改检索词。

## 暂停与恢复

运行中任务可暂停并退出。再次启动后，A11 任务从 SQLite 持久化 checkpoint 恢复，已正式提交的页面不会重新扫描。

Alpha10 已完成任务继续显示历史“约/約”检索条件。缺少可信页进度的旧未完成任务会显示 `LEGACY_TASK_REQUIRES_REVIEW`：旧结果保持只读，不会自动恢复或静默重扫；可使用任务页按钮把原目录带入新任务页面并重新扫描。

## 任务、诊断与本地数据

- “任务中心”可按任务名称、来源目录或检索词搜索，并按状态筛选和分页；从任一任务进入详情、校对或导出后，侧栏会保持该任务上下文。
- 左侧底部的“设置”页面集中管理应用配置。“校对显示与扫描上下文”提供 6 种预设高亮颜色、自定义颜色、10%～60% 透明度、4 种横排/竖排阅读方向，以及关键词每侧 1～50 字的上下文长度。高亮和上下文配置支持全局默认和指定任务覆盖；出处页始终按源文件无损显示。旧任务会按扫描时 SHA-256 按页升级，无可信指纹或原文件内容已变化时会要求重新扫描。原始文件不会被修改。
- 若任务已经创建但启动请求失败，任务会保留为“待启动”，详情页提供“启动任务”重试；无需返回新建页重复创建。
- “环境诊断”保留为独立页面，不在左侧主菜单展示；从“设置”页面点击“打开环境诊断”进入。诊断将 PASS/WARN/FAIL 转换为“可用/受限/不可用”，逐项显示实际影响、处理建议，并提供重新检查和打开日志目录入口。
- 仅缺少 Tesseract 二次复核、DJVU 工具或中文语言包时，整体环境显示“部分能力受限”；主识别引擎、ONNX Runtime、工作目录或本地识别服务不可用时，才显示“不可用”。
- 任务页可打开任务目录和应用日志。导出中心展示数据库中持久化的最近导出记录，重启应用后仍可查看。
- 当前版本不提供应用内任务删除或自动清理。若需手工清理，请先备份任务目录；不要在任务运行或校对保存期间删除文件。

## Alpha 限制

- 仅支持 Windows 10/11 x64。
- 安装包未签名，SmartScreen 可能显示未知发布者。
- PDF 主链路使用随包 pypdfium2/PDFium 与 RapidOCR；TIFF、JPEG、PNG 使用 Pillow 解码。
- 多页 TIFF 的每个 Frame 作为一个页面；JPEG、PNG 作为单页文档。图片单页最多 2 亿像素、最长边 30000 像素，多页 TIFF 最多 5000 页；不支持动态 PNG/APNG。
- 完整安装包已包含 Tesseract 二次复核、四个简繁中文语言包和 DJVU 工具；安装后无需另外下载。若包内文件损坏或缺失，诊断页会显示实际包内路径和处理建议。
- 文档与 OCR 结果默认只保存在本机，不上传网络。
